# Source Map

## Overview

Use this file to rank sources by confidence.
The goal is to keep the skill anchored to identifiable materials instead of internet folklore.

## 1. Primary Sources

List the most direct materials here:

- books written by the person
- letters
- diaries
- speeches
- interviews
- papers
- public memos
- verified transcripts

For each item, note what it is best for.

## 2. Secondary Sources

List biographies, scholarly studies, or credible long-form analysis here.
Use them to interpret the primary corpus, not to override it.

## 3. Low-Confidence Or Disputed Sources

List any popular sayings, contested books, edited collections, folklore, or viral quote lists here.
If a source is widely used but weakly supported, say so clearly.

## 4. Practical Rule

When building advice:

1. Start from the primary corpus.
2. Use secondary sources for context and organization.
3. Downgrade disputed materials.
4. Say when confidence is limited.
