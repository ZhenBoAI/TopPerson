# Principles

## Overview

Use this file to turn 孔子's recurring method into modern operating principles.
Prefer actionable paraphrase over quote worship.

## 1. 正名与角色责任

High-confidence basis:

- 《论语》与早期儒家文献反复讨论名分、角色与责任。

Modern translation:

- 先把我是谁、该做什么、不能越界什么讲清楚。

## 2. 仁恕与分寸

High-confidence basis:

- 仁、恕、礼是孔子处理中人与人关系的高频主题。

Modern translation:

- 处理关系先从尊重、克制和分寸感出发，而不是先争输赢。

## 3. 学而时习

High-confidence basis:

- 《论语》中对学习、复习、践行有持续强调。

Modern translation:

- 学习不能停在知道，要靠日常重复变成稳定习惯。

## 4. 反求诸己

High-confidence basis:

- 孔子式修身强调先修自己，再求外界配合。

Modern translation:

- 碰到问题先查自己哪里失礼、失责、失序。

## Scenario Mapping

### 角色混乱

- Diagnose: 冲突常来自责任未明、边界不清。
- First move: 先把每个人的角色、职责和期望写清楚。
- Avoid: 一上来就指责人格或动机。

### 学习停滞

- Diagnose: 问题不是知道得少，而是没有反复落实。
- First move: 先选一条最基础但最该践行的规则，连做七天。
- Avoid: 继续囤积新方法却不执行。

## Final Reminder

孔子式方法的现代价值，不是压人，而是让角色更清、关系更稳、自己更有分寸。
