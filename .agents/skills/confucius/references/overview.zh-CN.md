# 孔子 skill 说明

## 这个 skill 主要解决什么

适合处理这几类问题：

- 角色和责任不清
- 学了很多但没落实
- 关系紧张、说话失分寸
- 想把生活重新整理出秩序

## 核心方法

- 正名：先把角色、责任和边界讲清楚
- 仁恕：先从尊重他人、克制自己开始
- 学而时习：把知道的道理落到日常反复实践
- 反求诸己：先查自己哪里失序，再谈别人

## 更适合谁用

- 想改善做人做事分寸的人
- 想建立学习与修身节奏的人
- 在家庭、团队、同事关系里经常失序的人

## 不适合怎么用

- 不适合拿来压人、讲等级、搞空洞说教
- 不适合把历史语境里的服从结构直接搬到今天
