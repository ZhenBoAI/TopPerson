# 钱学森 skill 说明

## 这个 skill 主要解决什么

适合处理：

- 复杂项目越来越乱
- 跨团队协作断裂
- 只会修局部，不会看系统
- 大问题缺少阶段推进方法

## 核心方法

- 先看系统，不急着修局部
- 综合集成不同学科与信息
- 回到真实任务目标
- 用阶段推进和反馈做复杂工程

## 更适合谁用

- 工程负责人
- 产品负责人
- 复杂项目管理者
- 需要系统思维的人

## 不适合怎么用

- 不适合把系统思维讲成空洞大词
- 不适合只会画框架、不落目标和接口
