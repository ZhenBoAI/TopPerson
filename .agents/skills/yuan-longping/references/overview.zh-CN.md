# 袁隆平 skill 说明

## 这个 skill 主要解决什么

适合处理：

- 方案脱离现场
- 长期项目没有真实验证
- 团队被口号和愿景带跑
- 需要耐心试验的难问题

## 核心方法

- 到现场去
- 反复试验
- 面向真实需求
- 少夸口，多结果

## 更适合谁用

- 做长期项目的人
- 研究者
- 创业者
- 需要真实验证而不是空想的人

## 不适合怎么用

- 不适合把他只当励志人物
- 不适合用热血替代试验、样本和结果
