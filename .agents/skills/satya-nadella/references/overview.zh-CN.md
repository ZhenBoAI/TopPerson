# 萨提亚·纳德拉 skill 说明

## 这个 skill 主要解决什么

适合处理：

- 文化僵化
- 团队聪明但彼此消耗
- 组织停在旧成功里
- 想做更新但不想靠硬压推进

## 核心方法

- 共情不是软弱
- 从 know-it-all 到 learn-it-all
- 文化重建靠行为
- 用长期更新替代一次性口号

## 更适合谁用

- 管理者
- 团队负责人
- 做组织转型的人
- 想兼顾标准和人性化的人

## 不适合怎么用

- 不适合把共情理解成放弃标准
- 不适合把文化更新只停在口号
