# 维克多·弗兰克尔 skill 说明

## 这个 skill 主要解决什么

适合处理：

- 低谷里失去意义感
- 无法改变的痛苦
- 主体性变弱
- 想重新找回能承担的方向

## 核心方法

- 先找仍可承担的意义
- 环境再难，回应仍可选择
- 不神化痛苦
- 从具体责任恢复主体性

## 更适合谁用

- 正处在低谷的人
- 面临失去、挫折和无力感的人
- 想重新恢复行动感的人

## 不适合怎么用

- 不适合把苦难浪漫化
- 不适合把“意义”讲成空话或励志口号
